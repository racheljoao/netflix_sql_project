## Netflix SQL Dashboard Project
About the Project

For this project, I worked with a Netflix dataset to explore trends in movies and TV shows using SQL. I created queries to analyze things like content type, ratings, release years, and which countries produce the most Netflix content.

My main goal was to practice writing SQL queries and using data to answer business-style questions. I also used the results to build a dashboard that gives a simple visual overview of the dataset.
